# Driver Setup Guide

This guide explains how to set up and run the `backend` project on **Windows**.

---
---

```
cd ./Backend/python_engine
```

```
python -m venv venv
```

```
./venv/Script/Activate.ps1
```

```
pip install -r requirements.txt
```

---
---

```
cd ./Backend/node_server
```

```
npm i
```

```
node ./server.js
```
